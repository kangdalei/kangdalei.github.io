发表于 2020-10-18

windows下装机常用软件（备忘录）

今天（<2019-03-01 周五>）工作用电脑内存条坏了，趁机重装了一台电脑，升级了一下装备。不过很多软件就得重装了。下面分类记录：

[TOC]

浏览器

firefox 装机一般先装浏览器，需要它上网下东西。

键盘按键修改器 keymap

喜欢将ESC键和CapsLock键互换，将right_alt和right_ctrl互换。windows下通过修改注册表，不过把这堆代码记住很麻烦。

keyTweak 官网

输入法

rime 官网 配置好后，重装很简单。用户词库和输入习惯，自己备份一下。喜欢用带声调的地球拼音。

虚拟光驱

winmount 官网 好用不解释。

压缩软件

  • 7z 官网 经典老牌

  • Bandizip 官网 韩国人弄的，据说也不错。

搜索神器

  • Listary 官网 快速定位目录，搜索、启动器

  • everything 官网 按文件名闪电搜索

编程类

ssh客户端

MobaXterm 官网 free版已经很好用

git

https://git-scm.com/downloads

编辑器

notepad++ 官网

虚拟机

Oracle VM VirtualBox 官网

加密类

VeraCrypt 官网

翻译

  • CopyTranslator 官网 处理PDF文档很方便， 链结google翻译服务器。

  • GoldenDict 官网需要有好词典

电子文档处理

  • calibre 官网 文档管理全能解决方案

  • pandoc 官网 文档格式全能解决方案

  • sumatra pdf 官网 小巧玲珑。这软件和苏门达腊什么关系？

  • Icecream PDF Editor 官网 PDF文档编辑器。

  • cutepdf writer https://www.cutepdf.com/index.htm pdf虚拟打印机

  • Typora 官网 Markdown格式编辑器

  • cmd markdown 官网

  • Mathpix Snip 管网 公式图片转latex公式 Take a screenshot of math and paste the LaTeX into your editor, all with a single keyboard shortcut.

视频播放器

这类软件流氓很多，需要小心避免。

  • mplayer 官网

  • PotPlayer 官网 韩国的。不知为何网址被封。

下载软件

  • qbittorrent 官网 github上有搜索引擎目录 官网

  • aria2 官网 高冷命令行工具。 Persepolis Download Manager 图形化界面。

  • Motrix aria2 GUI封装

系统管理增强类工具

  • total commander

  • Ditto 剪贴板增强

  • spacesniffer 硬盘空间占用情况

  • Wiztree 速度更快的磁盘空间分析软件 仅支持 NTFS 下载

  • dngrep 官网 GUI版grep

  • dism++ 系统清理维护

截图贴图

  • snipaste 官网 一款截图贴图工具。截图大家都会，贴图就是杀手级特性了。
  • PicPick官网 便携版。支持滚动截屏。

录屏

  • ScreenToGif 官网 简单好用。截屏成gif动图或者视频(在FFmpeg支持下)。不过没声音。

  • Captura 官网 有声音！

  • obs-studio 官网 高端产品。同时录制屏幕和摄像头。没用过。

Emacs & keyboard 按键二三事

发表于 2017-10-13 更新于 2020-10-17

Emacs的按键是出了名的难按. 以至于新手入门第一件事就是改键盘布局, 否则没法学下去. 参考: Effective Emacs（中文）

难道不是很奇怪吗? 为何”神的编辑器”如此古怪难用? 其实这主要是因为Emacs远远早于 现代PC机键盘而出现,它的快捷键不是在PC机键盘上设计的, 而是在早已消失的Lisp机键 盘上敲定的. (参见: Peter Seibel 《编程人生》 Guy Steele 访谈篇)
